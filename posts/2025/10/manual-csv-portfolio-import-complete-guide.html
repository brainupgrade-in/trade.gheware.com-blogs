Broker-Specific Export Guides

Groww

  1. Login to Groww web platform
  2. Go to Stocks > Holdings
  3. Click "Download" icon (top-right)
  4. Select "Download as CSV"

Groww CSV has columns: Symbol, Company, Qty, Avg Cost

Upstox

  1. Login to Upstox Pro Web
  2. Navigate to Portfolio > Holdings
  3. Click "Export" button
  4. Choose CSV format

Upstox CSV includes: Scrip, Quantity, Average Price, Exchange

5paisa

  1. Login to 5paisa trading terminal
  2. Go to Reports > Holdings Report
  3. Click "Export to Excel"
  4. Save Excel as CSV format

5paisa provides Excel file - save as CSV before importing

Handling Large Portfolios (100+ Stocks)

Before Import

  • Clean data in Excel first
  • Remove duplicate entries
  • Standardize company names
  • Verify all symbols are correct

During Import

  • Split into 50-stock batches
  • Import sector-wise
  • Test with 5-10 stocks first
  • Keep backup of original CSV

Converting Excel to CSV

Using Microsoft Excel:

  1. Open your Excel file (.xlsx or .xls)
  2. Go to File > Save As
  3. In "Save as type", select "CSV (Comma delimited) (*.csv)"
  4. Click Save

Using Google Sheets:

  1. Open your spreadsheet in Google Sheets
  2. Go to File > Download
  3. Select "Comma-separated values (.csv)"

Frequently Asked Questions

What CSV format does Trade Gheware support?

Trade Gheware supports a 9-field CSV format. Required fields are: symbol, companyName, quantity, and averagePrice. Optional fields have default values.

Which brokers can I import from?

You can import from any broker that provides CSV export including Groww, Upstox, 5paisa, ICICI Direct, Angel One, HDFC Securities, and Sharekhan.

What happens if my CSV has validation errors?

Trade Gheware validates each row and shows specific error messages. Valid rows get green badges, invalid rows show red with error details. Fix errors and re-upload or import only valid rows.

What date format should I use?

Use the ISO 8601 date format: YYYY-MM-DD (e.g., 2024-01-15). The field is optional.
